London Housing Trend Dashboard
Overview
The London Housing Trend Dashboard analyzes housing prices and trends in London in detail. It is a dashboard suitable for interactively exploring  property prices across different aspects — neighborhood, property type, interior style, and building characteristics.

Key Features
1. Summary Metrics
Average Price of Housing in London: £1.84M. 
Average Price by Property Type:
Apartment: £1.47M
Semi-Detached: £1.82M
Detached House: £2.19M
2. Neighborhood Analysis
Top Neighborhoods:
Chelsea: £2.45M
Westminster: £2.29M
Kensington: £2.28M
Heatmap: See average price distribution in neighborhoods based on floor count and number of bedrooms.
3. Trends and Distributions
Price Trends by Interior Style and Building Age:
Compare different styles: Classic, Minimalist, Industrial, and Modern.
Price Distribution by Neighborhood and Type of Property:
See price distribution of Apartments, Semi-Detached Houses, and Detached Houses.
4. Top Properties
Most Expensive Houses: Top 5 properties costing between £7.22M and £8.05M.
Largest Houses: Top 5 by area from 548 m² to 739 m².
5. Categorized Price Distribution
Categorizes the properties as Budget, Mid-Range, and Luxury.
6. Export Options
Download insights and visualization to PNG files for offline usage.
